# codex-images-workflow

## Scope

Create and maintain reproducible generated-image artefacts with brief-first workflow, stable naming, per-image Markdown briefs, and explicit versioning. Use when Codex is asked to generate posters, mockups, concept art, UI screens, boards, or other project image assets that should be copied into a repo rather than left only in cache paths.

## Trigger

- Skill trigger: Manage reproducible generated-image artefacts.
- Procedure: see `SKILL.md`.
