# Agent Lessons

Escape hatch for feedback that does not compress into a single rule edit. Read on demand; not startup context. See `_AGENT-INSTRUCTIONS/self-healing.md` for the full flow.

## Rules

- Keep this index short.
- Add one topic file per lesson or lesson cluster.
- Link lessons from root `AGENTS.md` only when they are needed during common work.
- Use `_TASK-LOG/` for current status, blockers, and next action.
- Use `_CHANGELOG/` for decisions, backlog, and shipped changes.
- Use `README.md` for human background and related-project narrative.
- Revisit periodically: promote to a rule or retire.

## Status vocabulary

- `active` — still applies; not yet folded into rules.
- `promoted` — folded into `AGENTS.md` or a skill; file kept for audit trail.
- `superseded` — replaced by a newer lesson.
- `stale` — no longer accurate.
